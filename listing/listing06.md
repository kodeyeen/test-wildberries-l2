Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Выведется [3 2 3]

Инициализируется слайс из 3 строк: "1", "2" и "3"
Далее этот слайс передается в функцию modifySlice
На стеке функции modifySlice теперь есть копия слайса
Данная копия имеет те же поля, что и слайс из функции main
Т.е. тот же указатель на массив, та же длина и та же емкость
Далее мы меняем значение по индексу 0 в массиве, на который указывает указатель из структуры слайса
Таким образом, массив теперь хранит значения [3 2 3]
Потом мы в локальную переменную i переприсваиваем результат функции append
append возвращает новую структуру слайса
Т.к. у нашего слайса capacity был 3, то добавление нового 4-го элемента инициирует выделение нового массива
В общем, в i присвоится новая структура, в которой указатель на массив уже будет другой, т.к. при добавлении
нового элемента нам не хватило емкости
Дальше работа уже идет с новым слайсом (с новым массивом) и можно не рассматривать, что произойдет дальше,
т.к. мы уже можем ответить на вопрос, что выведет программа

Слайс это стуктура из 3 полей:
1) указатель на массив
2) len
3) cap
При передаче слайса в качестве аргумента функции слайс копируется на стек вызова функции
```
