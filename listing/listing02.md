Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Выведется "2 1"

defered функции исполняются ПОСЛЕ того, как возвращаемые значения установятся return statement'ом,
но ПЕРЕД тем, как управление вернется вызывающей программе

В defered функциях могут быть модифицированы только именованные результаты функции (как сказано в документации)
Анонимные результаты функций не могут быть модифицированы defered функциями, поэтому из anotherTest возвращается 1, а не 2

defered функции собираются в стек и исполняются в порядке LIFO (first in last out)
Этот стек реализован в виде связанного списка из структур _defer
У каждой структуры есть поле link *_defer (указатель на следующую структуру _defer)

```
